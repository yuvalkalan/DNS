# DNS
Basic IPv4 DNS server using python

## Commands

```bash
SHOW
```

show list of all urls and IPv4 address

```bash
SET <url address> <ip>
```

add url-ip entry

```bash
REMOVE <url address>
```

remove url-ip entry

```bash
SAVE
```

save changes locally

```bash
LOAD
```

load local changes

```bash
FIND <ip>
```

find url by ip

```bash
EXIT
```

exit program

```bash
HELP
```

show list of commands and parameters
